# Amandla.mobi WonderCMS theme

This is a (hopefully) simple theme for Amandla.mobi for use with [WonderCMS](https://www.wondercms.com). The goal is to replicate some of the current look and feel of [amandla.mobi](amandla.mobi) while keeping the total HTML+CSS+JS package extremely small, to better support viewers on mobile networks and with limited data plans.
